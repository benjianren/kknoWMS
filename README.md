# 智能补货模块 Demo 集合

## 文件说明

| 文件 | 说明 | 主用户 |
|------|------|--------|
| `P01_补货工作台.html` | 任务中枢，待办聚合 | 区域计划员 |
| `P03_补货策略编辑页.html` | 策略配置与试算 | 区域计划员、运营负责人 |
| `P04_补货建议池.html` | 建议处理列表 | 区域计划员 |
| `P06_预警中心.html` | 异常处理任务中心 | 区域计划员、仓内主管 |
| `P08_数据分析看板.html` | 价值证明与诊断 | 运营负责人、区域计划员 |

## 配色方案

基于 `frontend-style-guide.md` V1.2 版本的中国传统色：

| 颜色 | 色值 | 用途 |
|------|------|------|
| 天青 | `#6C9BCA` | 主色，按钮、链接、卡片标题 |
| 榴萼黄 | `#F9A633` | 辅助色，警告、强调 |
| 青绿 | `#53B886` | 成功状态 |
| 朱红 | `#C45C4A` | 错误/危险状态 |

## 页面预览

直接在浏览器中打开各 HTML 文件即可查看和交互，无需任何构建工具。

---

## P01 补货工作台

### 功能概览
- **关键待办卡片**：待确认建议、高风险建议、低可信库存、交期异常、待执行回写
- **今日优先处理对象**：按风险等级排序的待办列表
- **Top低可信库存对象**：库存可信度低的节点汇总
- **Top执行异常对象**：调拨失败、采购延迟等异常
- **快捷入口**：建议池、预警中心、执行跟踪、数据分析
- **趋势摘要**：近7天缺货率、采纳率、低可信库存数、待办完成率

---

## P03 补货策略编辑页

### 功能概览
- **基础信息**：策略名称、层级、说明
- **适用范围**：区域仓、品类、SKU选择
- **补货模式**：定量补货、定时补货、手工补货
- **参数配置**：采购交期、安全天数、目标覆盖天数、MOQ等
- **高级设置**：包装倍数、调拨优先
- **策略说明卡**：动态生成策略说明文案
- **公式说明卡**：业务语言+公式展开
- **样例SKU试算**：选择SKU执行试算，展示完整计算过程
- **覆盖/冲突提醒**：策略优先级和冲突提示

---

## P04 补货建议池

### 功能概览
- **顶部统计区**：今日新增、高风险、低可信、待审批、可优先调拨
- **Tab导航**：全部、待确认、高风险、低可信库存、调拨优先、采购优先、活动影响
- **筛选区**：区域仓、门店分组、品类、风险等级等
- **建议列表**：单号、节点、供给、类型、风险、可信度、建议量、来源策略
- **右侧快捷预览抽屉**：库存摘要、建议原因、操作按钮
- **底部批量操作栏**：批量提交审批、批量忽略、批量标记人工复核

---

## P06 预警中心

### 功能概览
- **顶部统计区**：今日新增、待处理、高优先级、低可信库存、交期异常
- **Tab导航**：全部、待处理、高优先级、低可信库存、交期异常、销量异常、执行异常
- **预警列表**：单号、类型、优先级、来源对象、状态、责任人、更新时间
- **右侧处理抽屉**：异常摘要、影响说明、建议动作、处理记录、操作按钮
- **底部批量操作栏**：批量认领、批量升级、批量关闭
- **异常类型**：低可信库存、采购交期异常、销量异常增长、调拨失败、执行回写异常

---

## P08 数据分析看板

### 功能概览
- **分析周期选择**：近7天、近30天、本月、自定义
- **核心指标卡**：缺货率、周转天数、建议采纳率、人工修正率、调拨替代占比
- **趋势图区**：缺货率趋势、周转天数趋势、采纳率趋势、低可信库存趋势
- **策略效果对比**：各策略的采纳率、修正率、缺货改善幅度
- **执行漏斗**：已生成→已确认→已审批→已执行→已完成回写
- **可信度影响分析**：高可信vs低可信库存对象的采纳率对比
- **明细榜单**：Top风险SKU、Top低效策略、Top未采纳建议

---

## 技术栈

- React 18 (CDN)
- Tailwind CSS (CDN)
- Font Awesome 图标
- Chart.js (P08趋势图)
- 单HTML文件，零配置，直接打开即可运行

## 配色说明

所有页面统一使用中国传统色配色方案：
- **天青 #6C9BCA**：主色调，用于主要按钮、链接、选中状态
- **榴萼黄 #F9A633**：辅助色，用于警告、强调、高亮
- **青绿 #53B886**：成功状态、正向指标
- **朱红 #C45C4A**：错误/危险状态、高风险提示

页面背景使用天青色调的浅渐变：`linear-gradient(180deg, #f8fafc 0%, #f0f7fc 100%)`
